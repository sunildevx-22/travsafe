# Travsafe
Travsafe is a full-stack web application designed to enhance traveler safety by providing real-time alerts, location-based information, and a seamless user experience.

# Features
- Real-time safety alerts using external APIs
- Location-aware notifications for travelers
- Responsive and user-friendly interface
- Secure and scalable backend architecture

# Tech Stack
- Frontend: React.js, HTML5, CSS3
- Backend: Node.js, Express.js
- Database: MongoDB

# Overview
Travsafe aims to provide users with timely and relevant safety information during travel. By integrating real-time APIs and building a scalable backend, the platform ensures efficient data handling and a smooth user experience across devices.

# Key Highlights
- Designed RESTful APIs for efficient communication between frontend and backend
- Structured MongoDB database for scalable data storage
- Implemented responsive UI for cross-device compatibility
